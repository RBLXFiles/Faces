# RBLXFiles - Roblox Files

Welcome to RBLXFiles. This is an open space focused on archiving, preserving, and providing historical Roblox files and assets for developers and enthusiasts.
## Roblox Face Archive

This project is a collection of Roblox face textures gathered from different eras of the platform. The repository serves as a historical archive for faces that have been removed, replaced, or are no longer officially distributed.

This collection includes:
* Classic Roblox faces.
* Faces that can no longer be obtained.
* Faces involved in hacking incidents.

## Guidelines and Contributions

Anyone may suggest the addition of missing face textures by opening an Issue or submitting a Pull Request on GitHub.

Please note the following guidelines:
* Only official Roblox face textures should be included.
* Edited, fan-made, or recreated faces should not be submitted.
* Images should be preserved in their highest available quality whenever possible.
* The purpose of this repository is historical preservation and documentation.

## Repository Structure

The archive is organized into folders based on the type or generation of Roblox faces.

Current folders include:
* **Classic** – Original Roblox face textures.

Additional categories may be added as more assets are archived.
